# factordb

`factordb` get the factorization of the number from factordb.com

### Usage

    USAGE:
        factordb [FLAGS] <NUM>

    FLAGS:
        -h, --help       Prints help information
        -r, --raw        Display the answer as raw json format
        -V, --version    Prints version information

    ARGS:
        <NUM>    Number to be factored
